# Arc Dark Theme for VS Code

A dark theme for Visual Studio Code based on the popular Arc Dark color scheme.

## Installation

1. Open VS Code
2. Go to Extensions (Cmd+Shift+X)
3. Search for "Arc Dark Theme"
4. Click Install
5. Go to Code > Preferences > Color Theme
6. Select "Arc Dark"

## How to Test

1.  Open the `arc-dark-vscode` folder in VS Code.
2.  Press `F5` to launch the Extension Development Host.
3.  In the new window, open the Command Palette (`Cmd+Shift+P`) and run `Preferences: Color Theme`.
4.  Select `Arc Dark`.
